User Management Overview

The User Module provides comprehensive functionality for managing users in the system, including creation, authentication, permissions, and profile management.

Key Features

  • Create, view, edit, and delete user accounts
  • Role-based access control with granular permissions
  • Bulk operations (import/export users)
  • User activity tracking

Viewing Users

To view the list of users in the system:

  1. Navigate to Users in the admin panel
  2. The system will display a paginated list of all users
  3. Use the search box to filter users by name, email, or other attributes


Creating Users

To create a new user:

  1. Click the Create User button
  2. Fill in the required fields:
    • Name of the user
    • NIC Number (must be valid and unique/ Enter without 'V' or 'X')
    • Email address (must be valid and unique)
    • Address & City
  3. Assign roles and permissions
  4. Click Save to create the user
Note: When sending an invitation email, the user will receive a link to set their own password. The password field can be left blank in this case.


Select User Roles



Editing Users

To modify an existing user account:

  1. Navigate to the user's profile
  2. Click the Edit button
  3. Make the necessary changes to the user's information
  4. Click Save to update the user
Note: When Editing a user profile, the password will be changed to default system password


Deleting Users

To remove a user from the system:

  1. Navigate to the user's profile
  2. Click the Delete button
  3. Confirm the deletion in the dialog box
Warning: Deleted users cannot be recovered. Consider deactivating instead of deleting if you might need the account later.


Exporting Users

To export user data:

  1. Navigate to the Users list
  2. Click the Export button
  3. Click Export to download the file
Note: You can Export users list as a Excel document.


Importing Users

To import users in bulk:

  1. Navigate to the Users list
  2. Click the Import button
  3. Upload your Excel file to the model
  4. Click Import to process the file
