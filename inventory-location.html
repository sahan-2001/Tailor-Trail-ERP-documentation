Inventory Location Module Documentation

Home / Inventory Modules / Inventory Location

Overview

The Inventory Location module helps organize storage by defining physical locations within warehouses.

Features

  • Assign locations to warehouses
  • Track items by location
  • Restrict access by roles

Viewing Inventory Locations

  1. Navigate to the Inventory Locations page from the sidebar
  2. A table will display all existing locations with their details
  3. You can search by location name, warehouse, or status


Creating a Location

  1. Go to the Inventory Locations page
  2. Click Create Inventory Location
  3. Fill in:
    • Location Name (e.g., Aisle 1-B)
    • Select Warehouse from dropdown
    • Optionally, add a Description
  4. Click Save to finish


Editing a Location

  1. Open the desired location
  2. Click Edit
  3. Update the name, warehouse, or description
  4. Click Save to apply changes


Deleting a Location

  1. Click the Delete button next to the location
  2. Confirm the deletion when prompted
Warning: Deleting a location may affect linked inventory items. Ensure it's no longer in use.


Exporting Inventory Locations

  1. Go to the Inventory Locations list page
  2. Click the Export button at the top right
  3. Select your desired format (e.g., Excel)
  4. Click Download to save the file
Note: Exported files include warehouse references and location descriptions.


Importing Inventory Locations

  1. Navigate to the Inventory Locations list
  2. Click the Import button in the top-right
  3. Upload your Excel file containing location data
  4. Click Import to complete the process
Note: Ensure the Excel file format matches the import template, including required fields like Location Name, Warehouse, and Status.
