Supplier Management Overview

This module helps manage suppliers, including their contact details, categories, and assigned materials or services.

Key Features

  • Add, edit, view, and delete suppliers
  • Make Payments for suppliers
  • Import and export supplier lists
  • Maintain location and contact info

Viewing Suppliers

Steps:

  1. Go to Suppliers tab
  2. List will show all existing suppliers
  3. Use filters or search to narrow down


Creating Suppliers

To create a new supplier:

  1. Click Create Supplier
  2. Enter:
    • Supplier Name, Shop Name
    • Email, Contact Number
    • City & Address
  3. Click Save
Note: Email and phone number should be unique.


Editing Suppliers

To edit a supplier:

  1. Select the supplier
  2. Click Edit
  3. Update necessary fields
  4. Click Save
Note: Ensure all required fields are filled.


Deleting Suppliers

To delete a supplier:

  1. Open the supplier profile
  2. Click Delete
  3. Confirm in popup
Warning: Deleted suppliers cannot be restored.


Exporting Suppliers

Steps:

  1. Go to the supplier list
  2. Click Export
  3. Download the Excel file
Note: You can export for backups or sharing.


Importing Suppliers

To bulk import:

  1. Click Import
  2. Upload Excel template
  3. Click Import again to confirm
Note: Use provided sample Excel format.
