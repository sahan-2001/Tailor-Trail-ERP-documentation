Warehouse Overview

The Warehouse Module allows creating and managing warehouse facilities with capacity, location, and notes.

Key Features

  • Create, view, edit, and delete warehouse records
  • Capture location and dimension data
  • Apply filters based on ID, name, or measurement unit
  • Role-based permission for actions

Viewing Warehouses

  1. Navigate to Inventory Management > Warehouses
  2. The warehouse list will be displayed with ID, name, capacity, and unit
  3. Use filters to narrow down results


Creating Warehouses

  1. Click Create Warehouse
  2. Fill in required details:
    • Warehouse Name
    • Address (3 lines) and City
    • Length, Width, Height, and Unit of measurement
    • Optional Notes
  3. Click Save


Editing Warehouses

  1. Select a warehouse from the list
  2. Click the Edit icon
  3. Update the relevant fields
  4. Click Save to apply changes
Note: Name must remain unique after editing.


Deleting Warehouses

  1. Click the Delete icon for the selected warehouse
  2. Confirm deletion in the dialog popup
Warning: This action is permanent. Deleted records cannot be restored.


Exporting Warehouses

To export warehouse data:

  1. Navigate to the Warehouses page
  2. Click the Export button in the top-right or toolbar
  3. Select your preferred file format (e.g., Excel, CSV)
  4. Click Export to download the data
Note: Only visible columns and filters will be included in the export file.
