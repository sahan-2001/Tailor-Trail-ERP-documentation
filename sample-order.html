Sample Order Module

Home / Order Modules / Sample Orders

Overview

The Sample Order module manages all customer sample orders, allowing tracking from creation to fulfillment with item variations and detailed workflows.

Key Features

Feature Description
Customer Linked Orders Select customers with auto-filled contact details
Item Variations Supports order items with multiple variations and pricing
Status Workflow Manage orders through planned, released, completed, invoiced, and other states
Filters and Search Filter orders by status, creation date, or delivery date
Role-Based Actions Actions like edit, delete, and handle orders based on permissions and order status

Viewing Sample Orders

To browse and search sample orders:

  1. Go to Orders > Sample Customer Orders
  2. The table lists all sample orders with sortable columns
  3. Key columns include:
    • Order ID - Unique identifier (5-digit padded)
    • Customer Name - Linked customer record
    • Order Name - Descriptive order title
    • Wanted Delivery Date - Requested delivery date
    • Status - Current order state
    • Total Sale - Grand total amount
  4. Use search and filters to narrow down results
  5. Click column headers to sort orders
Tip: Use the "Handle" action to manage an order’s lifecycle and process changes.


Available Filters

  • Status — Filter orders by their current state
  • Created Date — Filter orders by creation date
  • Wanted Delivery Date — Filter by requested delivery date

Creating a Sample Order

To create a new sample order:

  1. Click Create Sample Order in the sample orders section
  2. Fill out the form fields:

    Customer Details

    • Customer — Search and select from existing customers
    • Customer details (name, phone, email) auto-populate

    Order Details

    • Order Name — Descriptive title for the sample order
    • Wanted Delivery Date — Requested delivery date (not before today)
    • Special Notes — Optional instructions or comments

    Order Items

    • Add items using the Add Order Item button
    • For each item:
      • Item Name — Product description
      • Is Variation — Toggle to enable multiple variation items
      • If no variation:
        • Quantity
        • Price
        • Total (auto-calculated)
      • If variation enabled:
        • Add sub-items using Add Variation Item
        • Each variation requires name, quantity, price, and auto-calculated total
  3. Click Save to create the order
Note: The grand total automatically calculates from all items and variations.


Editing a Sample Order

To edit an existing sample order:

  1. Find the order in the list
  2. Click the Edit button (pencil icon)
  3. Modify allowed fields:
    • Order Name
    • Wanted Delivery Date
    • Special Notes
    • Order Items — quantities, prices, additions, or removals
  4. Click Save to update the order
Restriction: Orders can only be edited while in "planned" or "released" status.


Deleting a Sample Order

To delete a sample order:

  1. Locate the order in the list
  2. Click the Delete button (trash icon)
  3. Confirm the deletion in the popup
Important: Only orders in "planned" status can be deleted. This action is irreversible.


Handling Sample Orders

The handling interface allows processing orders through their lifecycle:

  1. Click the Handle action in the orders list
  2. Review order details and available actions
  3. Possible actions based on status:
    • Release Order (planned → released)
    • Complete Order (released → completed)
    • Mark as Delivered (completed → delivered)
    • Cancel Order (planned → cancelled)
    • Print PDF (available at any stage)
Audit Trail: All status changes are logged with user and timestamp details.

Status Definitions

Status Description Allowed Actions
Planned Order created but not yet processed Edit, Delete, Release
Released Order approved and ready for processing Complete, Cancel, Print
Completed Order process finished Mark as Delivered, Print
Delivered Order delivered to customer Accept, Reject, Print
Accepted Order accepted by the customer Convert, Print
Rejected Order rejected by the customer Print
Converted Order Converted as a customer order Go to customer order, Print
Cancelled Order cancelled Print
PDF Generation: You can generate professional PDFs of sample orders at any stage for records or customer sharing.
