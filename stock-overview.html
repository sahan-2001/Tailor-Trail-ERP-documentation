Stock Overview Module

Home / Inventory Modules / Stock Overview

Overview

The Stock Overview module provides comprehensive management of inventory stock levels across warehouses and locations, with detailed tracking of all stock movements.

Key Features

Feature Description
Stock Tracking Real-time visibility of stock quantities by warehouse and location
Stock Movements Record retrievals, returns, and destructions with audit trails
Purchase Order Integration Special handling for PO-related stock with return capabilities
Bulk Operations Import/export stock data for efficient inventory management
Advanced Filtering Filter by warehouse, location, item code, and more
Audit Logging Track all changes with user and timestamp information

Viewing Stock Records

To browse and search current stock levels:

  1. Navigate to Inventory > Stock Overview
  2. The system displays all active stock records (quantity > 0)
  3. Key information shown:
    • Warehouse - Storage facility
    • Location - Specific bin/shelf location
    • Item Code - Unique item identifier
    • Item Name - Product description
    • Available Quantity - Current stock level
  4. Use the search bar to find specific items
  5. Click column headers to sort records
  6. Apply filters to narrow results (see below)
Tip: The system only shows stock with positive quantities. Zero-quantity items are automatically hidden.

Available Filters

  • Item Code - Filter by full or partial item codes
  • Warehouse - Show stock from specific warehouses
  • Location - Filter by picking/shipment locations


Creating Stock Records

To add new stock to your inventory:

  1. Click Create Stock from the stock overview page
  2. Complete the form sections:

    User & Timestamps (Auto-filled)

    • User ID - Your account identifier
    • User Email - Your email address
    • Received Date - Defaults to current date
    • Updated Date - Auto-managed by system

    Stock Details (Required)

    • Item - Search and select inventory item
    • Available Quantity - Auto-populated from item
    • Quantity - Amount being added (must be ≥1)
    • Cost - Unit cost for accounting
    • Location - Select picking location
  3. Click Save to create the stock record
Important: The quantity you enter will be added to any existing stock of the same item at the selected location.


Retrieving Stock

To remove stock from inventory (for use, transfer, etc.):

  1. Locate the stock record in the overview
  2. Click the Retrieve Stock action button
  3. The form auto-populates with:
    • Item details (ID, code, name)
    • Location information
    • Current available quantity
    • Current cost (editable if needed)
  4. Enter:
    • Quantity to Retrieve (must be ≤ available)
    • Reason - Mandatory explanation
  5. Click Submit to complete the retrieval
Audit Trail: All retrievals are logged with user, timestamp, and reason for complete traceability.


Returning Stock (PO Items Only)

To return stock that came from a purchase order:

  1. Locate the PO-related stock record (marked with PO icon)
  2. Click the Return Stock action button
  3. The form includes:
    • Original PO information
    • Supplier/Customer details
    • Current stock details
  4. Specify:
    • Quantity to Return (must be ≤ available)
    • Reason - Mandatory explanation
  5. Click Submit to process the return
Restriction: This feature is only available for stock records linked to purchase orders.


Destroying Stock

To permanently remove damaged or expired stock:

  1. Locate the stock record in the overview
  2. Click the Destroy Stock action button
  3. The form auto-populates with current stock details
  4. Enter:
    • Quantity to Destroy (must be ≤ available)
    • Reason - Mandatory explanation
  5. Click Submit to confirm destruction
Irreversible Action: Destroyed stock cannot be recovered. Ensure this is the correct action before proceeding.


Exporting Stock Data

To export stock records for reporting or backup:

  1. From the stock overview, click Export
  2. Select format:
    • Excel - Full data with formatting
    • CSV - Simple comma-separated values
    • PDF - Printable report format
  3. Choose columns to include (default: all visible columns)
  4. Apply any active filters to export filtered subsets
  5. Click Export to download the file
Template: Exported Excel files can serve as templates for imports - just add new rows with your data.


Importing Stock Data

To bulk import stock records from a spreadsheet:

  1. Prepare your data file:
    • Use the exported template or follow required format
    • Required fields: item_id, quantity, location_id
    • Recommended fields: cost, received_date
  2. From the stock overview, click Import
  3. Upload your prepared file
  4. Map spreadsheet columns to database fields
  5. Review the preview for validation errors
  6. Click Import to complete the process
Validation Rules:
  • Items must exist in inventory
  • Locations must be valid picking locations
  • Quantities must be positive numbers
  • Duplicate entries will be skipped
