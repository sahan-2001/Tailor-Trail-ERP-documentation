Purchase Order Overview

The Purchase Order Module facilitates procurement management by handling orders with suppliers and customers in a unified workflow.

Key Features

  • Create, view, edit, delete purchase orders
  • Supports supplier and customer as order providers
  • Add multiple order items with quantity and price
  • Status workflow: planned, released, paused, completed
  • Order handling interface with lifecycle actions
  • Generate PDF invoices and order documents

Viewing Purchase Orders

  1. Navigate to Order Modules > Purchase Orders
  2. All purchase orders display in a sortable, searchable table
  3. Columns include: Order ID, Provider Type, Provider ID, Wanted Date, Status
  4. Use filters to refine the list by provider type and wanted date
  5. Click column headers to sort
  6. Click on order rows for more details or actions


Creating a Purchase Order

  1. Click Create Purchase Order button
  2. Fill in required fields:
    • Select Provider Type (Supplier or Customer)
    • Search and select the Provider
    • Set Wanted Delivery Date
    • Optionally add Special Notes
  3. Under the Order Items section, add items:
    • Select inventory item
    • Specify quantity and price
    • Use "Add Order Item" to include multiple items
  4. Click Save to create the order
Note: Provider Type and Provider cannot be changed after order creation to ensure consistency.


Editing a Purchase Order

  1. Find the order in the purchase orders list
  2. Click the Edit icon to open the order form
  3. Modify allowed fields such as:
    • Wanted Delivery Date
    • Special Notes
    • Order Items (add, edit, or remove)
  4. Click Save to apply changes
Warning: Only orders in "planned" status can be edited.


Deleting a Purchase Order

  1. Locate the order in the list
  2. Click the Delete icon
  3. Confirm deletion in the popup dialog
Warning: Only "planned" status orders can be deleted. This action is permanent.


Handling Purchase Orders

The handling interface allows managing the order lifecycle with actions:

  • Release Order: Change status from planned to released
  • Plan Again: Move back to planned from released or paused
  • Pause Order: Temporarily pause a released order
  • Resume Order: Resume a paused order
  • Print PDF: Generate printable order documents


Status Workflow

Purchase orders follow this status flow:

1
Planned
2
Released
3
Paused
4
Completed

Status Definitions

Status Description Allowed Actions
Planned Order created, pending processing Edit, Delete, Release
Released Order approved and sent to provider Plan Again, Pause, Print
Paused Order temporarily on hold Resume, Plan Again, Print
Completed Order fulfilled and closed Print
Note: Orders can be printed as PDF documents at any stage of the workflow.
